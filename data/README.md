Generated data files are organized here.

- `extracts/`: raw extracted task snapshots from ClickUp (`*_folder_tasks_*.json`).
- `outputs/`: analyzed pipeline outputs consumed by dashboard APIs (`output_*.json`).
- `ams/`: AMS classification artifacts (`ams_tasks.json`, `ams_classified*.json`, `ams_blocker_analysis.json`).

Dashboard APIs keep backward-compatible fallbacks to legacy root/dashboard locations.
